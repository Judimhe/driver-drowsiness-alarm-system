# Driver Drowsiness Alarm System

An Arduino-based safety system designed to detect driver inactivity and trigger alerts to reduce accidents caused by drowsiness.

## Features
- Driver activity monitoring
- PIR motion sensor detection
- Alarm and buzzer alerts
- Automatic system reset
- Tinkercad circuit simulation

## Components Used
- Arduino Uno
- PIR Sensor
- Buzzer
- LED
- Breadboard
- Push Button

## Technologies Used
- Arduino C++
- Digital Logic Design
- Tinkercad Simulation

## Future Improvements
- LCD status display
- Camera-based eye detection
- Mobile notifications
- Vehicle control integration
